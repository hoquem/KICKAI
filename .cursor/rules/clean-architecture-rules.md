## üèóÔ∏è Architecture Layers and Boundaries

### Layer Definitions

```
üì± Application Layer
‚îú‚îÄ‚îÄ Purpose: Framework integration and external interfaces
‚îú‚îÄ‚îÄ Contents: @tool decorators, command handlers, API controllers
‚îú‚îÄ‚îÄ Dependencies: Domain Layer only
‚îî‚îÄ‚îÄ Forbidden: Business logic, direct database access

üß† Domain Layer  
‚îú‚îÄ‚îÄ Purpose: Pure business logic and domain models
‚îú‚îÄ‚îÄ Contents: Entities, services, repository interfaces, domain rules
‚îú‚îÄ‚îÄ Dependencies: None (center of architecture)
‚îî‚îÄ‚îÄ Forbidden: Framework imports, external service calls

üîß Infrastructure Layer
‚îú‚îÄ‚îÄ Purpose: External concerns and implementation details
‚îú‚îÄ‚îÄ Contents: Database implementations, external APIs, file systems
‚îú‚îÄ‚îÄ Dependencies: Domain Layer interfaces only
‚îî‚îÄ‚îÄ Forbidden: Business logic, direct application coupling
```

### Dependency Rules (MANDATORY)

```
Application Layer ‚îÄ‚îÄdepends on‚îÄ‚îÄ> Domain Layer
Infrastructure Layer ‚îÄ‚îÄimplements‚îÄ‚îÄ> Domain Layer Interfaces
Domain Layer ‚îÄ‚îÄdepends on‚îÄ‚îÄ> NOTHING EXTERNAL

‚úÖ ALLOWED: Application ‚Üí Domain
‚úÖ ALLOWED: Infrastructure ‚Üí Domain (via interfaces)
‚ùå FORBIDDEN: Domain ‚Üí Application
‚ùå FORBIDDEN: Domain ‚Üí Infrastructure
‚ùå FORBIDDEN: Application ‚Üî Infrastructure (direct coupling)
```

### Rule 4: Entity and Value Object Purity

**‚úÖ CORRECT - Domain Entity:**
```python
# kickai/features/player_registration/domain/entities/player.py
from dataclasses import dataclass
from typing import Optional, Dict, Any
from datetime import datetime

@dataclass
class Player:
    telegram_id: int
    team_id: str
    name: str
    position: Optional[str] = None
    created_at: Optional[datetime] = None
    
    @classmethod
    def from_dict(cls, data: Dict[str, Any]) -> 'Player':
        """Create from dictionary data (e.g., from database)."""
        return cls(
            telegram_id=data.get('telegram_id'),
            team_id=data.get('team_id'),
            name=data.get('name'),
            position=data.get('position'),
            created_at=data.get('created_at')
        )
    
    def to_dict(self) -> Dict[str, Any]:
        """Convert to dictionary for persistence."""
        return {
            'telegram_id': self.telegram_id,
            'team_id': self.team_id,
            'name': self.name,
            'position': self.position,
            'created_at': self.created_at.isoformat() if self.created_at else None
        }
```

### Architecture Compliance Tests

```python
# tests/architecture/test_clean_architecture_compliance.py
import ast
import pytest
from pathlib import Path

class TestCleanArchitectureCompliance:
    def test_domain_layer_purity(self):
        """Ensure domain layer has no framework dependencies."""
        domain_files = Path('kickai/features').glob('*/domain/**/*.py')
        violations = []
        
        for file_path in domain_files:
            violations.extend(self._check_imports(file_path))
        
        assert not violations, f"Framework imports found in domain layer: {violations}"
    
    def test_application_layer_delegation(self):
        """Ensure application tools delegate to domain layer."""
        app_tools = Path('kickai/features').glob('*/application/tools/*.py')
        
        for tool_file in app_tools:
            with open(tool_file, 'r') as f:
                content = f.read()
            
            # Should have @tool decorators
            assert '@tool(' in content, f"Missing @tool decorator in {tool_file}"
            
            # Should delegate to domain layer
            assert 'domain' in content, f"No domain delegation in {tool_file}"
    
    def _check_imports(self, file_path):
        """Check file for forbidden framework imports."""
        forbidden_modules = ['crewai', 'firebase_admin', 'telegram']
        violations = []
        
        try:
            with open(file_path, 'r') as f:
                tree = ast.parse(f.read())
                
            for node in ast.walk(tree):
                if isinstance(node, ast.ImportFrom) and node.module:
                    for forbidden in forbidden_modules:
                        if forbidden in node.module:
                            violations.append(f"{file_path}: {node.module}")
        except Exception:
            pass  # Skip files that can't be parsed
            
        return violations
```
